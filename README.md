# Jagruti Popali - Personal Portfolio Website

A modern, responsive, and multi-page portfolio website developed using **HTML5** and **CSS3**. This portfolio showcases my education, technical skills, internships, projects, and contact information in a professional and visually appealing format.


## About the Project

This portfolio website was created to represent my journey as a Computer Science Engineering student and aspiring web developer. It highlights my academic background, technical skills, internship experiences, and projects while providing an easy way to connect with me.

The website follows modern design principles with a clean dark theme, responsive layouts, and smooth navigation across multiple pages.
